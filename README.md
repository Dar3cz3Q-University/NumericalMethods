## Setup

### Local development
1. Clone the repository:
   ```shell
   git clone --recursive https://github.com/Dar3cz3Q-University/NumericalMethods.git
   cd NumericalMethods
   ```
2. Run script in [Scripts](/Scripts) directory
3. Open Visual Studio solution
